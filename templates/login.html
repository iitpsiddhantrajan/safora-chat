Welcome to Safora

Safora is a real-time secure messaging platform developed by Siddhant Rajan, an undergraduate student at IIT Patna.

This project demonstrates full-stack engineering skills including:

  • Secure authentication with password hashing
  • Real-time private messaging using WebSockets
  • Online / Offline presence tracking
  • Typing indicators
  • Message delivery acknowledgments

Built using Python, Flask, Socket.IO, SQLite, and modern UI design principles.
